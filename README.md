# Text Translator

Simple Python scripts for translating texts from English and Italian into Russian.

## Features
- Translate English -> Russian
- Translate Italian -> Russian
- Works with plain text files

## Technologies
- Python 3
- Googletrans library (Python library for Google Translate API)

## How to Use
1. Clone the repository
2. Install dependencies: `pip install googletrans==4.0.0rc1`
3. Run the script: `python translator.py`
4. Input text or file to translate
